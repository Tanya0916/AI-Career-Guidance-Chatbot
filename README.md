# 🤖 AI Career Guidance Chatbot

An AI-powered chatbot built using **Python**, **NLTK**, **Scikit-learn**, and **Streamlit** that provides career guidance through Natural Language Processing (NLP). The chatbot recognizes user intent and responds with relevant career advice, interview tips, resume guidance, certification suggestions, and more.

---

##  Project Overview

This project demonstrates how Natural Language Processing (NLP) can be used to build an intelligent conversational chatbot. The chatbot classifies user queries into predefined intents and returns dynamic responses related to career guidance.

## 🧠 NLP Pipeline

1. Convert text to lowercase
2. Tokenization
3. Remove punctuation
4. Remove stopwords
5. Lemmatization
6. CountVectorizer
7. Intent Classification
8. Response Generation

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Career-Guidance-Chatbot.git
```

Move into the project

```bash
cd AI-Career-Guidance-Chatbot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Train the chatbot

```bash
python train_model.py
```

Run the application

```bash
streamlit run app.py
```
